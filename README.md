# CountDownProgressBar
首次提交

这是一个带有，圆环动画的倒计时控件，很常用。

使用：

allprojects {
		repositories {
			...
			maven { url 'https://www.jitpack.io' }
		}
	}

implementation 'com.github.xuanfengwuxiang:CountDownProgressBar:1.1'
